# Web Elements Automation – Selenium Practice Website

This project automates the website:
https://testautomationpractice.blogspot.com

The site is designed for testing practice and contains many UI elements, making it ideal for demonstrating Selenium skills.

## What the automation covers
The scripts interact with a wide range of web elements, including:
- Input fields
- Buttons
- Radio buttons and checkboxes
- Dropdowns and lists
- Alerts
- Frames and iframes
- Multiple windows
- Tables
- Drag & drop
- Upload / download actions
- Dynamic elements

## Technologies Used
- Python  
- Selenium WebDriver  
- ChromeDriver  

## Folder Structure
- **Scripts/** – Full Selenium automation scripts  
- **Screenshots/** – Before and after execution screenshots  

## Purpose of the Project
This project demonstrates:
- Hands-on experience automating many different element types  
- Locators, waits, assertions, and element handling  
- Ability to automate a full practice website  
- Strong understanding of core UI automation concepts
